Работа кода на страницах типа default.

ОБЩИЙ КОД

- загрузка route.php
    - узнать у сервера ip посетителя
    - передать этот ip функции isBot()
        - узнать по ip адрес хоста
        - по хосту сделать вывод : бот это или нет
        - вернуть результь true|false
    - если true
        - отправить данные о заходе бота на почту
            - узнать по ip адрес хоста
            - по хосту определит тип бота
            - узнать адрес по ip
            - узнать страницу посещения
            - составить письмо с этими данными
            - отправить письмо на почту
    - если false
        - считать настройки фильтров из файла config.json
        - если фильтр по ip : TRUE
            - передать ip и узнать город
            - определить тип устройства
            - если город Москва, Томск и это планшет или компьютер (кроме моих ip - добавь сюда Анапский ip)
                - перенаправь на страницу 404
                - выйди

- рендеринг страницы
    - блок loading (про нормы воды) : ON, на всю страницу
    - блок closed (ошибка 404) : OFF
    - блок open (контент) : OFF

- загрузка скриптов

- скрипт useConfig()
    - скрыть блок loading
    - запрос на сервер : считать файл config.json, узнать значения фильтров
    - если только на телефонах
        - назначь свойство show-for-small-only блоку open
    - если сайт ON
        - показать блок open
    - если сайт OFF
        - блок open скрыть
        - блок closed показать

ИНДИВИДУАЛЬНЫЙ КОД

- запроси с сервера данные ip посетителя : сам ip и адрес ip, сохрани в Local Storage
